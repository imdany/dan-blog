About me

Daniel is a Data Engineer / Solution Architect mainly focused on data architectures and data processing projects.

With more than seven years of experience, Daniel has worked on a multitude of projects for big and small companies, implementing and designing data solutions that cover:

  • Extraction and Processing of data from Streaming and Batch sources
  • Storage of the data in Data Lakes / Data Warehouses
  • Extraction of insights from the data using Analytics and Data Science Techniques
  • Implementing reporting and dashboard solutions
  • Implementing APIs for exposing data
  • Setting up the infrastructure and services (on-prem and cloud architectures)

Programming languages: Python, Scala, Java, R

Main Techonologies:

  • Data Ingestion and Data Processing pipelines (Spark (Databricks), Flink, Dask, Pandas)
  • Data Warehouse / Data Lake Design (PostgreSQL, SQL Server, Oracle) / Azure Data Lake, S3, Hadoop
  • Reporting solutions (PowerBI, Microstrategy)
  • Data Science / Machine Learning (SKLearn, Keras)
  • Kubernetes / Docker
  • DevOps (Jenkins, Azure DevOps) / MLOps
  • Software Engineering / Testing (TDD, BDD)
  • API Design
  • Cloud Technologies (Azure, AWS)

Certifications
